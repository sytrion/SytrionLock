### About
This is a multi-platform username checker tool. It helps you check if a username is available or already taken on many popular social media platforms.

### Features
- Supports 20+ popular platforms
- Beautiful colorful interface
- Shows popular account examples
- Random similar username suggestions
- Easy to use

### Supported Platforms

Most Popular:
Instagram, TikTok, X (Twitter), YouTube, Telegram, Snapchat, Facebook, Roblox, Twitch, Reddit

Others:
Spotify, Pinterest, LinkedIn, Messenger, Discord, Netflix, CapCut, Zoom, WhatsApp, Gmail
